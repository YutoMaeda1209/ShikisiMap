# 敷嶋てとら聖地巡礼スポット登録ガイドライン
本ドキュメントでは、敷嶋てとらファンメイド聖地巡礼マップ「ShikisiMap」に、スポットを追加する際のガイドラインについて説明します。

## 目的
- 利用者に有益な情報を提供する
- データの重複や誤情報を避け、メンテナンスしやすくする

## 追加の可否判断
- [敷嶋てとら・敷嶋重工 二次創作ガイドライン](https://tetratex01.my.canva.site/guideline)を遵守していること
- 敷嶋てとらが関係するコンテンツ内で実際に訪れている場所
- 敷嶋てとらが十分に言及していること
- 有料コンテンツなど、無制限にアクセスできないコンテンツ以外で言及されていること

## 品質基準
スポット追加時には以下の品質基準を満たしていることを確認してください。
- 正確性：位置情報はGoogle MapsやOpenStreetMapなど信頼できるソースから取得し、閉業情報は複数の情報を確認すること
- 有益性：敷嶋てとらコンテンツ内で十分に言及されており、かつ訪問済みであること
- 合法性：追加時点で立ち入り禁止区域、または私有地でないこと

## 引用動画の選定基準
複数の動画で同じスポットが登場する場合、以下の基準で引用動画を選定してください。
- 敷嶋てとらがスポットに初めて訪れた動画
- 敷嶋てとらがスポットに最も詳しく言及している動画

## フォーマット
スポットは`src/data.json`に[GeoJSON](https://datatracker.ietf.org/doc/html/rfc7946)形式で追加します。
下記のフィールド一覧に従って追記してください。
- `type`: string - `Feature`固定
- `properties`
  - `name`: string - スポット名
  - `address` : string - スポットの正確な住所
  - `isClosed`: boolean - 現在閉業している場合は`true`
  - `youtubeId`: string - スポットが登場するYouTube動画のID
  - `timestamp`: string - スポットが登場するYouTube動画内のタイムスタンプ（秒）
- `geometry`
  - `type`: string - `Point`固定
  - `coordinates`: array[number] - `address`の座標（[経度, 緯度]の順）

## レビュー手順
1. `src/data.json`にスポットを追加する
1. [コントリビューションガイドライン](/.github/CONTRIBUTING.md)に従って、Pull Requestを作成する
1. 一名以上のレビュアーが掲載妥当性、フォーマット、情報を確認する
1. 承認後にマージする
